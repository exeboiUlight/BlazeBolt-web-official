Возможности

Всё для 2D-игры в одном скачивании

BlazeBolt предоставляет полный набор инструментов для создания 2D-игр — от спрайтов и анимаций до физики и сетевого взаимодействия.

🎨

Спрайты и анимации

Спрайты с текстурными атласами, анимированные спрайты (GIF), AnimationWheel для переключения состояний.

⚙️

Собственная физика

Собственный физический движок: статические, кинематические и динамические тела, коллайдеры, силы, импульсы.

🌕

Система частиц

GPU-инстансированная система частиц с настраиваемым временем жизни, скоростью, размером, цветом и направлением.

🌈

Освещение

Точечные и фоновые источники света с настраиваемым цветом, интенсивностью и радиусом.

🎬

Кастомные шейдеры

Поддержка пользовательских вершинных и фрагментных шейдеров через Mesh-объекты с uniform-переменными.

🎧

Аудио (OpenAL)

Загрузка звуков, 3D-позиционирование, спикер, EFX-эффекты (реверберация и др.), фильтры.

🌐

Сеть (TCP/UDP)

Встроенный сетевой стек: TCP/UDP серверы и клиенты для многопользовательских игр.

🎲

Тайловые карты

Tileset2D с поддержкой текстурных атласов, установкой тайлов по сетке.

📈

Процедурные шумы

Перлин, симплекс, Value Noise, FBM и Domain Warp для генерации миров и эффектов.

💾

Математические типы

Vector2/3/4, Matrix3x3 с операторами и методами для работы с трансформациями.

📁

Сцены и редактор

Формат .scene (JSON), хаб/редактор с инспектором, Gizmo, загрузка и сохранение сцен.

📝

Lua API

Полный Lua 5.4 скриптинг с lifecycle-коллбеками (Start/Update/Draw/End), загрузка скриптов на лету.
