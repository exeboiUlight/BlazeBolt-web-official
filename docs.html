🔥 BlazeBolt Engine

Lua API Reference — Complete Documentation

Version 1.2 · Lua 5.4 · OpenGL 3.3

Overview

BlazeBolt Engine — 2D engine with Lua 5.4 scripting, OpenGL 3.3 rendering, custom physics engine, audio, networking functions and procedural noise generation.

All engine functions are available through the global table BlazeBolt:

local sprite = BlazeBolt.CreateSprite("texture.png", 100, 200)
BlazeBolt.SpriteSetPosition(sprite, 300, 400)

Some functions are registered as globals (without the BlazeBolt table):

local w = GetScreenWidth()
local h = GetScreenHeight()
local n = PerlinNoise2D(x, y)

Entity type — is an integer identifier returned by entity creation functions.

Lifecycle calls Lua functions Start(), Update(dt), Draw(), End() automatically.

Data types: number (float), integer, string, boolean, nil, table, Entity (integer), Vector2, Vector3, Vector4, Matrix3x3.

Lifecycle (Callbacks)

The engine automatically calls certain Lua functions at different stages:

CallbackDescription
Start()Called once at launch
Update(dt)Called every frame. dt — time since last frame (seconds)
Draw()Called every frame for rendering
End()Called on shutdown
On<SceneName>Load()Called when scene is loaded
On<SceneName>Unload()Called when scene is unloaded
local player
function Start()
    player = BlazeBolt.CreateSprite("player.png", 100, 100)
end
function Update(dt)
    local x, y = BlazeBolt.SpriteGetPosition(player)
    BlazeBolt.SpriteSetPosition(player, x + 100 * dt, y)
end
function Draw() -- rendering happens automatically end

Example: Sprites

local player
function Start()
    player = BlazeBolt.CreateSprite("assets/player.png", 100, 200)
    BlazeBolt.SpriteSetOrigin(player, 0.5, 0.5)
    BlazeBolt.SpriteSetSize(player, 64, 64)
    BlazeBolt.SpriteSetRotation(player, 45)
    BlazeBolt.SpriteSetColor(player, 1.0, 1.0, 1.0, 0.8)
end
function Update(dt)
    local x, y = BlazeBolt.SpriteGetPosition(player)
    if BlazeBolt.IsKeyPressed(Keys.RIGHT) then x = x + 200 * dt end
    if BlazeBolt.IsKeyPressed(Keys.LEFT) then x = x - 200 * dt end
    BlazeBolt.SpriteSetPosition(player, x, y)
    local r = BlazeBolt.SpriteGetRotation(player)
    BlazeBolt.SpriteSetRotation(player, r + 90 * dt)
end
function Draw()
    if BlazeBolt.IsKeyPressed(Keys.H) then BlazeBolt.SpriteSetVisible(player, false) end
end

Example: SpriteBatch

local batch
function Start()
    batch = BlazeBolt.CreateSpriteBatch(100)
    BlazeBolt.SpriteBatchSetTexture(batch, "assets/grass.png")
    for i = 0, 99 do

        BlazeBolt.SpriteBatchAdd(batch, s)
    end
end

Example: AnimationWheel

local sprite, wheel
function Start()
    sprite = BlazeBolt.CreateAnimatedSprite("idle.gif", 400, 300)
    wheel = BlazeBolt.CreateAnimationWheel(sprite)
    BlazeBolt.AnimationWheelAddState(wheel, "idle", "idle.gif", 1.0, true)
    BlazeBolt.AnimationWheelAddState(wheel, "run", "run.gif", 1.5, true)
    BlazeBolt.AnimationWheelAddState(wheel, "jump", "jump.gif", 1.0, false)
    BlazeBolt.AnimationWheelSetInitialState(wheel, "idle")
    BlazeBolt.AnimationWheelSetState(wheel, "idle")
end
function Update(dt)
    if BlazeBolt.IsKeyJustPressed(Keys.ONE) then BlazeBolt.AnimationWheelSetState(wheel, "idle")
    elseif BlazeBolt.IsKeyJustPressed(Keys.TWO) then BlazeBolt.AnimationWheelSetState(wheel, "run")
    elseif BlazeBolt.IsKeyJustPressed(Keys.THREE) then BlazeBolt.AnimationWheelSetState(wheel, "jump") end
end

function Start()

    local mesh = BlazeBolt.CreateMesh()
    local vertices = {
        { x=0, y=0, u=0, v=0 }, { x=w, y=0, u=1, v=0 },
        { x=w, y=h, u=1, v=1 }, { x=0, y=h, u=0, v=1 },

Scene Management

LoadScene / GetCurrentScene

BlazeBolt.LoadScene(sceneName) → success BlazeBolt.GetCurrentScene() → sceneName

Example: Scenes

function Start() BlazeBolt.LoadScene("main_menu") end
function Update(dt)
    local s = BlazeBolt.GetCurrentScene()
    if s == "main_menu" and BlazeBolt.IsKeyJustPressed(Keys.ENTER) then BlazeBolt.LoadScene("game") end
    if s == "game" and BlazeBolt.IsKeyJustPressed(Keys.ESCAPE) then BlazeBolt.LoadScene("main_menu") end
end

Scene File Format (.scene)

Scene files use JSON format with an array of objects. Each object describes one scene element.

Document Structure

{ "name": "my_level", "version": 1, "objects": [...] }

Supported Types

Each object has a type field: sprite, animated_sprite, text, camera, point_light, ambient_light, particle_system, tileset, physics_body.

Physics Body Properties

Any object can include physics by adding a body_type field:

FieldTypeDescription
body_typenumber0=Static, 1=Dynamic, 2=Kinematic (omit for no physics)
massnumberMass (default 1.0)
frictionnumberFriction (default 0.3)
restitutionnumberRestitution / bounciness (default 0.5)
collider_shapestring"circle" or "rectangle"
circle_radiusnumberCircle radius (default 32)
circle_offset_x/ynumberCircle offset from object position
rect_half_width/heightnumberHalf-size for rectangle collider (default 32)
gravity_scalenumberGravity scale (default 1.0)
fixed_rotationboolLock rotation
bulletboolContinuous collision detection

Example: Sprite with Physics

{
    "objects": [
        {
            "name": "player",
            "type": "sprite",
            "texture": "player.png",
            "pos_x": 100,
            "pos_y": 200,
            "body_type": 1,
            "mass": 1.0,
            "collider_shape": "circle",
            "circle_radius": 32
        },
        {
            "name": "wall",
            "type": "physics_body",
            "body_type": 0,
            "collider_shape": "rectangle",
            "rect_half_width": 200,
            "rect_half_height": 20
        }
    ]
}

Example: Input

local player, speed = nil, 200
function Start() player = BlazeBolt.CreateSprite("assets/player.png", 400, 300) end
function Update(dt)
    local x, y = BlazeBolt.SpriteGetPosition(player)
    if BlazeBolt.IsKeyPressed(Keys.W) then y = y - speed * dt end
    if BlazeBolt.IsKeyPressed(Keys.S) then y = y + speed * dt end
    if BlazeBolt.IsKeyPressed(Keys.A) then x = x - speed * dt end
    if BlazeBolt.IsKeyPressed(Keys.D) then x = x + speed * dt end
    BlazeBolt.SpriteSetPosition(player, x, y)
end

Noise

Procedural noise generators. All functions are globals (not in BlazeBolt table).

Noise Functions

SetNoiseSeed(seed) PerlinNoise1D(x) → value PerlinNoise2D(x, y) → value PerlinNoise3D(x, y, z) → value SimplexNoise2D(x, y) → value ValueNoise2D(x, y) → value FbmNoise2D(x, y, [octaves], [lacunarity], [gain]) → value FbmSimplexNoise2D(x, y, [octaves], [lacunarity], [gain]) → value DomainWarpNoise2D(x, y, [warpScale]) → value

All return values from -1 to 1. Default octaves=6, lacunarity=2.0, gain=0.5.

Example: Noise

local mesh
function Start()
    SetNoiseSeed(42)
    mesh = BlazeBolt.CreateMesh()
    local w = GetScreenWidth(), h = GetScreenHeight()
    BlazeBolt.MeshSetData(mesh,
        {{x=0,y=0,u=0,v=0},{x=w,y=0,u=1,v=0},
         {x=w,y=h,u=1,v=1},{x=0,y=h,u=0,v=1}},
        {0,1,2,0,2,3})
    BlazeBolt.MeshSetShader(mesh, "shaders/terrain.vert", "shaders/terrain.frag")
end

Example: TCP Networking

local server, client
function Start()
    BlazeBolt.NetInit()
    server = BlazeBolt.CreateTCPServer(8080)
    client = BlazeBolt.CreateTCPClient()
    BlazeBolt.TCPClientConnect(client, "127.0.0.1", 8080)
end
function Update(dt)
    BlazeBolt.TCPServerPoll(server)
    if BlazeBolt.IsKeyJustPressed(Keys.SPACE) then
        BlazeBolt.TCPClientSend(client, "Hello from client!")
    end
end
function End()
    BlazeBolt.TCPServerStop(server)
    BlazeBolt.TCPClientDisconnect(client)
    BlazeBolt.NetShutdown()
end

Example: UDP Networking

local server, client
function Start()
    BlazeBolt.NetInit()
    server = BlazeBolt.CreateUDPServer(9090)
    client = BlazeBolt.CreateUDPClient()
    BlazeBolt.UDPClientConnect(client, "127.0.0.1", 9090)
end
function Update(dt)
    BlazeBolt.UDPServerPoll(server)
    if BlazeBolt.IsKeyJustPressed(Keys.SPACE) then BlazeBolt.UDPClientSend(client, "ping") end
    local peerId, data = BlazeBolt.UDPServerReceiveAny(server)
    if peerId then BlazeBolt.UDPServerSend(server, peerId, "pong") end
end

Entity Destruction

Destroy

BlazeBolt.Destroy(entity)

DestroyAll

BlazeBolt.DestroyAll()

Example: Destruction

local entities = {}
function Start()
    for i = 1, 10 do
        entities[i] = BlazeBolt.CreateSprite("assets/star.png", i * 50, 300)
    end
end
function Update(dt)
    if BlazeBolt.IsKeyJustPressed(Keys.ONE) and #entities > 0 then
        BlazeBolt.Destroy(entities[#entities])
        entities[#entities] = nil
    end
    if BlazeBolt.IsKeyJustPressed(Keys.TWO) then BlazeBolt.DestroyAll() entities = {} end
end
